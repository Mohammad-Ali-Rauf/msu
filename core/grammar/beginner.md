# **MSU Standard Urdu Grammar – Beginner**

---

## **1. Nouns (اسم)**

### **1.1 Gender (جنس)**

Urdu nouns have **two genders**:

* **مذکر (Masculine)** → لڑکا، کتا، استاد
* **مؤنث (Feminine)** → لڑکی، بلی، استادہ

### **1.2 Number (تعداد)**

* **واحد (Singular)** → ایک واحد شخص یا چیز

  * Example: کتاب، لڑکا
* **جمع (Plural)** → زیادہ افراد یا چیزیں

  * Example: کتابیں، لڑکے، لڑکیاں

### **1.3 Rules**

* Masculine nouns ending in "ا" → plural often "ے"

  * لڑکا → لڑکے
* Feminine nouns ending in "ی" → plural often "یاں"

  * لڑکی → لڑکیاں

---

## **2. Pronouns (ضمیر)**

| Type          | Singular | Plural          | Notes            |
| ------------- | -------- | --------------- | ---------------- |
| First person  | میں      | ہم              | “I / We”         |
| Second person | تم / آپ  | تم لوگ / آپ لوگ | Polite vs casual |
| Third person  | وہ       | وہ لوگ          | He/She/They      |

**Examples:**

* میں پڑھتا ہوں۔
* تم کھاؤ۔
* وہ کھیل رہا ہے۔

---

## **3. Verbs (فعل)**

### **3.1 Tenses (زمانہ)**

#### Present (حال)

* Structure: Subject + Object + Verb
* Examples:

  * علی کھاتا ہے۔
  * لڑکیاں کھیلتی ہیں۔

#### Past (ماضی)

* Structure: Subject + "نے" + Object + Verb
* Examples:

  * علی نے کتاب پڑھی۔
  * لڑکوں نے کھانا کھایا۔

#### Future (مستقبل)

* Verb stem + گا / گی / گے
* Examples:

  * علی جائے گا۔
  * لڑکیاں آئیں گی۔

#### **Future Negation**

* Place **نہیں** before the main verb:

  * علی نہیں جائے گا۔
  * ہم نہیں کھائیں گے۔

---

### **3.2 Verb Conjugation Table**

| Subject | Verb (to eat) | Translation            |
| ------- | ------------- | ---------------------- |
| لڑکا    | کھاتا ہے      | He eats                |
| لڑکی    | کھاتی ہے      | She eats               |
| لڑکے    | کھاتے ہیں     | They (boys) eat        |
| لڑکیاں  | کھاتی ہیں     | They (girls) eat |

> **Note:** Always match verb gender & number with the subject.

---

### **3.3 Negation (نفی)**

* Place **نہیں** before the main verb:

  * علی نہیں کھاتا۔
  * ہم نے کام نہیں کیا۔
  * Future: علی نہیں جائے گا۔

---

### **3.4 Imperative (حکم)**

* **Casual Singular**: کھاؤ، پڑھو، آؤ
* **Casual Plural**: In informal speech, plural and singular imperatives often sound the same. Context and tone help distinguish them.
* **Polite/Formal**: آئیں، کریں، دیکھیں

  * Example: آپ آئیں، آپ کھائیں

---

## **4. Adjectives (صفت)**

* Adjectives agree in **gender and number** with the noun:

  * اچھا لڑکا → اچھی لڑکی → اچھے لڑکے → اچھی لڑکیاں

**Placement:** usually before the noun

* Example: بڑا گھر، نئی کتاب

---

## **5. Postpositions (حرفِ جر)**

Urdu uses **postpositions** (after the noun) instead of prepositions.

| Postposition | Meaning    | Example     |
| ------------ | ---------- | ----------- |
| کا / کی / کے | possession | علی کی کتاب |
| میں          | location   | اسکول میں   |
| سے           | from/by    | دوست سے     |
| پر           | on         | کتاب پر     |
| کے ساتھ      | with       | علی کے ساتھ |

---

## **6. Sentence Structure (جملے کا ڈھانچہ)**

### **6.1 Basic Order**

* Urdu is **SOV (Subject Object Verb)**:

  * علی نے کتاب پڑھی۔

* **Negative**: Subject + Object + نہیں + Verb

  * علی نے کتاب نہیں پڑھی۔

### **6.2 Questions (سوالیہ جملہ)**

* **Yes/No Questions**: Use intonation or **کیا**

  * کیا علی آیا؟

* **WH-Questions**: Use question word

  * کون آیا؟
  * کہاں گیا؟
  * کب کھایا؟

---

## **7. Conjunctions (حروفِ ربط)**

| Word | Meaning | Example                       |
| ---- | ------- | ----------------------------- |
| اور  | and     | علی اور فہد کھیل رہے ہیں۔     |
| لیکن | but     | علی اچھا ہے لیکن فہد بہتر ہے۔ |
| یا   | or      | چائے یا کافی؟                 |

## **8. Common Particles (چھوٹے الفاظ)**

| Word | Usage    | Example           |
| ---- | -------- | ----------------- |
| ہی   | emphasis | یہی کتاب ہے۔      |
| بھی  | also     | علی بھی آیا۔      |
| تو   | then/so  | تم نے تو آنا تھا۔ |

---

## **9. Numbers (اعداد)**

* Basic numbers: ایک، دو، تین، چار، پانچ…
* Agreement:

  * دو لڑکے → دو لڑکیاں

---

## **10. Common Mistakes to Avoid**

* ❌ Using masculine verbs with feminine nouns:

  * لڑکی پڑھتا ہے۔ → ✅ لڑکی پڑھتی ہے

* ❌ Mixing postpositions incorrectly:

  * علی کتاب میں۔ → ✅ علی کتاب میں ہے

* ❌ Forgetting auxiliary “نے” in past tense:

  * علی کتاب پڑھی۔ → ✅ علی نے کتاب پڑھی
